Try Moscow ML

Warning: Try Moscow ML is currently under heavy development, it may change dramatically, the server may crash or be hacked, or your interactions and code may be recorded and logged. Thus, use it with care.

This is simple web-based environment for experimenting with SML code.

Editor keybindings

Press Ctrl+b to send the content of the editor to the toplevel.

Console keybindings

Use arrows for history.

Press Ctrl+c to interrupt the interpreter *not working, yet).

Try Moscow ML by @kfl, concept inspired by Try Haskell, editor powered by CodeMirror, and console powered by jquery-console.
